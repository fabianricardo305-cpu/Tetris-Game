# Tetris Game

A Tetris implementation built with Python and Pygame.

## Features
- Classic Tetris gameplay
- Mouse-controlled horizontal movement
- Keyboard rotation (R key)
- Fast drop (S key)
- Handles complex row-clearing edge cases (rows cleared between blocks)

## Installation

1. Install the latest version of Python
2. Install Pygame-ce

## How to Run
Execute through the main.py file.

## Controls
- **Mouse**: Move piece left/right
- **R**: Rotate piece
- **S**: Fast drop

## Project Structure
- main.py- Main game logic
- blcks.py- Tetromino shape classes
